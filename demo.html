Gallery - image viewer library

Linking to the library 


        <link href="path/to/Gallery/style.css" rel="stylesheet" />
        <script src="path/to/Gallery/script.js"></script>

How to use

For minimal use, you need to specify a container for the gallery and a list of images.

The gallery will take up all the free space in the container.

Parameters are passed to the Gallery function as a JSON object.

Example 1 


        Gallery({
          wrapper: ".wrapper",
          images: [
           "images/image1.jpg",
           "images/image2.jpg",
           "images/image3.jpg",
           "images/image4.jpg",
           "images/image5.jpg",
           "images/image6.jpg",
           "images/image7.jpg",
           "images/image8.jpg"],
        });

Available settings

zoom - image enlargement

grab - moving image

fullsize - full screen view

strictfullsize - built-in browsers full screen view

counter - information about the total number of images and the number of the current image

center - the ability to center the image after dragging or resizing

*If any parameter is not specified explicitly, it is considered true by default.
